# ASCII Art Generator

一个有趣的 ASCII 艺术生成器，支持多种字体、颜色和交互功能。

## 功能

- 🎭 **艺术字体** - 使用 pyfiglet 生成多种风格的 ASCII 艺术字
- 🐱 **迷你动物艺术** - 10种可爱的 ASCII 动物
- 🔮 **魔法8球** - 摇一摇获取答案
- 🔮 **算命先生** - 东方神秘算命体验
- 🎮 **石头剪刀布** - 跟电脑对战
- 💕 **爱情计算器** - 测算两人缘分
- 🐄 **ASCII 奶牛** - 会说话的 ASCII 奶牛
- 💬 **随机语录** - 程序员名言警句
- 🌈 **彩虹文字** - 彩色文字输出
- ⏳ **旋转动画** - 酷炫加载动画

## 安装

```bash
# 克隆项目
git clone <repository-url>
cd python-tutorials

# 创建并激活 Python 3.12 环境（推荐）
conda create -n py312 python=3.12 -y
conda activate py312

# 安装依赖
pip install pyfiglet
```

## 运行

```bash
python3 ascii_art_generator.py
```
